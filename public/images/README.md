# OG Image

Place your Open Graph image here as `og-image.jpg`.

Recommended specifications:
- Dimensions: 1200 x 630 px
- Format: JPG or PNG
- Max file size: 8 MB
- Content: Doyen Auto Services logo, garage name, and a tagline
- It will appear when pages are shared on Facebook, Twitter/X, LinkedIn, WhatsApp etc.

This file is referenced in:
- resources/views/app.blade.php
- resources/js/Pages/Landing.vue
- resources/js/Pages/About.vue
